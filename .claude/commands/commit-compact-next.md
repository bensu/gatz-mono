Use the /commit command under `.claude/commands/commit.md` to commit the current changes related to the task you are working on. If you finished the task or the spec, mark it as done and include it in the commit.

Once the commit is over, do a /compact

Once the /compact is over, use `.claude/commands/work-on-task.md` to work on the next task for this spec. If there are no other tasks for the spec, mark the spec as done.
